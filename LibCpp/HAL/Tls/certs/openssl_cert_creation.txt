Configure trusted root CAs for OpenSSL:
see: https://stackoverflow.com/questions/25156180/how-to-list-certificates-trusted-by-openssl
Download trusted CA .pem file from https://curl.se/docs/caextract.html
Rename the file to cert.pem and place it to the foulder: C:\Program Files\Common Files\SSL
You can find this foulder by: openssl version -d
